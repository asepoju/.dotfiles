OS X
========
### Setup
1.  Apple ID 생성. 카드번호가 필요하며, 구버전 OS X의 경우 iTunes를 써서 활성화해야한다.
1.  OS 업데이트
1.  키 반복속도, 마우스 속도 설정
1.  Finder 초기 디렉토리 경로 변경
1.  [Customize <kbd>caps lock</kbd>](https://pqrs.org/osx/karabiner/faq.html.en#capslock)
